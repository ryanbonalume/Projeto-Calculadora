### GIT
Aprendendo comandos Git

Comando para inicializar um repositorio:
```
git init
```
Comando para ver o Status da branch:
```
git status
```
Comando para adicionar alterações:
```
git add .
```
Comando para fazer um Commit:
```
git commit -m "nome-do-commit"
```
